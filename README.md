# Self Hosted Email Server (WIP)

A self-hosted email system built with:

- Docker Mailserver (Postfix + Dovecot)
- Go backend (SMTP + IMAP)
- Gin API layer

## Features (Current)

- Send emails via SMTP
- Fetch emails via IMAP
- REST API:
  - POST /send-email
  - GET /emails

## Status

🚧 Work in progress — building a full open-source alternative to email hosting services.

## Goal

Allow users to:

- Attach their own domain
- Create email accounts
- Send & receive emails
- Fully control their infrastructure (privacy-first)

## Tech Stack

- Go
- Docker
- Postfix
- Dovecot
